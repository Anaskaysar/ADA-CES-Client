## Run Project Client Side

Commands: 

### Clone
``` 
git clone --branch dashboard https://github.com/Anaskaysar/ADA-CES-Client.git
```
## Env File

` copy env.txt to root directory and rename it .env.local `

### Install
``` 
npm i
```

### Run

```
npm run
```
